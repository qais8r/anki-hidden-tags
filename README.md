# Hidden Tags

Hide tags from the Browser sidebar without changing note/card tag data.

## Usage

1. Open the Browser.
2. In the sidebar tag tree, right-click a tag and choose `Hide Tag`.
3. Open `Tools > Hidden Tags` to unhide selected tags or all hidden tags.

## Notes

- Uses full tag paths as keys (for example: `A::B::C`).
- Affects Browser sidebar display only.
- Does not delete, rename, or modify tags on notes/cards.
