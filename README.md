# Hidden Tags

Install the Anki addon from [AnkiWeb](https://ankiweb.net/shared/info/1129879202).

## Description

Hide tags from the Browser sidebar without changing or deleting note/card tag data.

![Demo GIF showing the process of hiding and unhiding a tag](https://github.com/qais8r/anki-hidden-tags/blob/main/assets/demo.gif?raw=true)

## Usage

1. Open the Browser.
2. In the sidebar tag tree, right-click a tag and choose `Hide Tag`.
3. Open `Tools > Hidden Tags` to unhide selected tags or all hidden tags.

## Notes

- Uses full tag paths as keys (for example: `A::B::C`).
- Affects Browser sidebar display only.
- Does not delete, rename, or modify tags on notes/cards.
